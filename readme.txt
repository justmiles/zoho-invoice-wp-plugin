Upload `zoho-invoice.zip` onto your local computer.2. Go to your WordPress Dashboard and select <strong>Plugins >> Add New</strong>.3. Click on the <strong>Upload</strong> option at the top and select the `zoho-invoice.zip file` you just downloaded.4. Click on <strong>Install</strong>.5. Activate the plugin through the 'Plugins' menu in WordPress6. There should be an additional Zoho Invoice sub-menu under the Settings option of your Dashboard.7. Add your Auth token and API key to the settings menu.8. Execute Zoho Invoice via shortcodes. == Frequently Asked Questions ==== Screenshots ==== Changelog ==BUG: Cosmetic
